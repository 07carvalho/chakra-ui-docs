## Overview

The Modal component is a multipart component. Read more about how to style
multipart components in the
[Component Style](https://chakra-ui.com/docs/styled-system/component-style#styling-multipart-components)
page.

## Anatomy

- `header`
- `overlay`
- `dialogContainer`
- `dialog`
- `closeButton`
- `body`
- `footer`

You can find more information in the source for the styles
[here](https://github.com/chakra-ui/chakra-ui/blob/main/packages/components/theme/src/components/modal.ts).

## Defaults

The default `size` is `md`
